    ☰ 카테고리

    e-편한서비스 회원 탈퇴는 로그인한 상태에서

    회원님이 직접 진행해야 합니다.

    ★ 회원 탈퇴 절차

    ㅇ Web

    1. e-편한서비스 웹서비스의 왼쪽 바의 MyPage를 클릭합니다.

    2. 클릭 시, 토글 버튼아래의 Leave 버튼을 클릭합니다.

    3. 해당 페이지에서 '회원탈퇴' 버튼을 클릭합니다.


    ㅇ 회원 탈퇴 안내사항

    e-편한서비스를 탈퇴하면 자사서비스 관련 정보제공 이용에 제한이 될 수 있습니다.

    - 고객님의 개인정보는 원칙적으로 탈퇴 즉시 파기됩니다.

    (다만 개인정보 보관기관에 대해 고객님의 별도 동의를 얻은 경우 또는 법령에서 저촉되지 않는 경우에 안전하게 보관 후 파기 됩니다.)

    * 수집 시에 동의받은 개인정보 보유 기간은 회원 탈퇴후 5년까지

    ** 탈퇴 후 e-편한서비스는 언제든 재가입 할 수 있습니다.
